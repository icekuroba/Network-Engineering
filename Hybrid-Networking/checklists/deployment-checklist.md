# Deployment Checklist

## Before Deployment

- [ ] Confirm on-premises IP ranges.
- [ ] Confirm cloud IP ranges.
- [ ] Verify that no subnets overlap.
- [ ] Define VPN peer public IPs.
- [ ] Define routing requirements.
- [ ] Define firewall rules.
- [ ] Define cloud security rules.
- [ ] Document management access.
- [ ] Validate DNS requirements.

## After Deployment

- [ ] Verify VPN tunnel status.
- [ ] Test ping or TCP connectivity where allowed.
- [ ] Confirm routes on both sides.
- [ ] Confirm firewall logs.
- [ ] Confirm cloud security rule logs.
- [ ] Validate application connectivity.
- [ ] Validate management access restrictions.
